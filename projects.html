Proyectos

Proyecto 1: Aplicación de Clima

Descripción: Una aplicación web que proporciona información sobre el clima en tiempo real utilizando una API pública. Los usuarios pueden buscar el clima actual en diferentes ubicaciones. Tecnologías Utilizadas: HTML, CSS, JavaScript, API de OpenWeatherMap Características: Muestra la temperatura, condiciones meteorológicas y pronóstico a corto plazo. Funcionalidad de búsqueda para diferentes ciudades. Diseño amigable y fácil de usar.

Proyecto 2: Portfolio Personal / DDS

Descripción: Un sitio web personal diseñado para mostrar proyectos, habilidades y experiencias profesionales. Incluye secciones para información personal, habilidades, proyectos y contacto. Tecnologías Utilizadas: HTML, CSS, Bootstrap, JavaScript Características: Diseño responsive que se adapta a diferentes tamaños de pantalla. Secciones para habilidades, proyectos y formulario de contacto. Integración con Google Fonts para mejorar la tipografía.
