Habilidades

HTML5 & CSS3

Utilizo HTML5 y CSS3 para construir interfaces de usuario limpias y responsivas. Mi enfoque está en crear diseños que sean accesibles y visualmente atractivos en todos los dispositivos.

JavaScript

Con JavaScript, añado interactividad y dinamismo a los sitios web. Desde validación de formularios hasta la creación de aplicaciones web interactivas, JavaScript es fundamental para mejorar la experiencia del usuario.

React

Uso React para desarrollar aplicaciones web eficientes y de alto rendimiento. Gracias a su arquitectura basada en componentes, puedo construir interfaces de usuario escalables y mantenibles.

Node.js

Node.js me permite crear aplicaciones del lado del servidor rápidas y escalables. Lo utilizo para desarrollar API RESTful y gestionar la lógica de backend de mis proyectos web.

Python

Python es mi herramienta de elección para el análisis de datos y la automatización de tareas. Utilizo Python para desarrollar scripts y aplicaciones que resuelven problemas complejos y optimizan flujos de trabajo.
